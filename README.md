# hello_world
a simple cpp code that outputs time and the text "hello_world" to a file specified in the command line arg

# compile
## Windows 
- download MSYS2 and gcc
- set the env variable to the gcc downloaded by MSYS2
- g++ hello_world.cpp

## Linux
- install gcc
- g++ hello_world.cpp

# usage
- `hello_world <path_to_log_file>`
